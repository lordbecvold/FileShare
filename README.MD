## About FileShare

Simple file sharing app

## Installation
 - Edit config.php
 - Set web root to public/
 - Grant read/write permission to web user

 ## Dependencies
This project requires PHP8 or < .
* Plupload
   * Version: **1.5.8**
   * [Website](https://www.plupload.com/)
